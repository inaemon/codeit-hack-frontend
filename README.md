## 📍 필수 기능
1. 회원가입/로그인에 SDK 활용
2. 지도
3. 리뷰/평점
4. 달력
5. 반응형 웹

</br></br>
--- 
## 📍 주요 기능
1. 구글 아이디로 로그인 가능
2. 구글 지도로 장소 검색
3. 캘린더로 여행 날짜 선택
4. 구글 리뷰 뿐만 아니라 자체 리뷰 서비스 제공
5. 여행 스케줄 계획</br>
6. 반응형 웹을 사용하여 모바일/웹 어느 환경에서나 최적화된 화면 제공</br>
